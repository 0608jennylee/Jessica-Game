## To download the project for prototype:
### Requirements:
Unity version 2018.1.2f1
### Execution instructions:
#### Download contents of Prototype folder and:

1) Run Prototype.exe

#### OR build from unity:

1) clone https://github.com/rayyoung122/306-Project-2 onto a local directory
2) pull from the repository (master branch)
3) open the cloned folder in Unity
4) go to Assets/Scenes
5) double-click to open WelcomeScene
6) run WelcomeScene to play game

#### OR run release from

1) https://github.com/rayyoung122/306-Project-2/releases/tag/v1.1
2) download and unzip the Prototype.zip
3) run Prototype.exe

## Game Walkthrough ##

_Note: This is the most efficient to complete each level. It does not guarantee receiving the highest score or full completion of the achievements. Any deviation from these instructions may result in gameplay being completed before being able to finish the game._  

### Level 1 Walkthrough ###

Starting scene

   * Walk north into the school building.  
   
Hallway

   * Go to the hallway on the left, avoiding or battling the NPC  
   
Classroom  

   * Talk to the teacher at the west side of the room.
   * Travel east to the hallway, and east again to the library
   
Library

* Read the three books closest to the east side.
* Return to the classroom by travelling west and west again.
* Classroom 2
* Talk to the teach on the east side of the room
   * Answer: Yes
   * Answer: Programming OR Electronics Technology
* Travel east to the hallway, south to go outside, then west to the field

Field

* Avoid or battle the NPC, and pick up the key in the north west area of the map
* Walk east, then north to enter the hallway again.
* Walk north to the computer lab, then west to the mentor’s lab

Mentor’s lab

* Talk to the professor at the east side of the room.

### Level 2 Walkthrough ###  
	
Opening Scene (Seminar Room)

* Select “we’ll be fine” option
* Go to next room (to hallway)

Hallway 

* Select ECE upon entering (or use cheat code to skip all activities related to the 3 engineering departments, jump to *)
* Go to the computer lab ( ↓ that way)

Computer Lab

* Talk to Kevin (guy with a thunder on the shirt) (could use cheat code to skip the next part)
* Play games on the 2 computer on the left 
   * Answer: the gas pedal
   * Answer: set an exact time for the meeting
* Talk to Kevin again
* Go to hallway (hallway ↑)

Hallway

* Talk to Amy (girl in the hallway ↑)
* Select Civil 
* Go to study area (on the right →)

Study area
* Talk to Alice (girl in pink dress)(could use cheat code to skip the whole activity)
* Talk to the two boys 
* Select “Ask if they need help” option 
* Talk to Alice again
* Go back to hallway (on the left ←)

Hallway
* Choose “mechanical” option
* Go upstair (using the green stairs in the room) 

Upstairs
* Go to the room on the right, talk to NPC in the room (Alex) (could use cheat code to skip the activity)
* Get white item from the library (go through computer lab, and then go to the room on the left) 
* Talk to Alex 
* Go back to hallway 

Hallway
* *Talk to Amy 
* Go upstair, go to the room on the left

WEN room 
* Talk to NPC

### Level 3 Walkthrough ###

<p align="center">
  <img src ="https://github.com/rayyoung122/306-Project-2/blob/master/images/README/level-3-map.PNG"/>
</p>

Opening scene(Player will be in the Reception)

Use cheat code to skip part 1 for marking purposes: 

* Go to Sam’s Office
* Talk to Sam
* Click yes to use cheat code
* Proceed to part 2

**Part 1: Collecting items**

* Go to Office - interact with TODO list,
* Go to Misc - interact with Printer (you will get some paper in the item slot)
* Go to Sam’s office - interact with Sam and lose paper
* Go to Misc - interact with coffee
* Go back to Sam office - interact with Sam

Use cheat code to skip part 2 for marking purposes: 

* Go to Reception - talk to Receptionist Beryl
* Interact with laptop
* Click yes for use cheat code
* Choose Winning condition or losing condition
* Go to Sam
* If chose losing condition then go back to laptop
* If chose winning condition then go to Sophia
* Proceed to part 3

**Part 2: Allocating people**

* Go to Reception - talk to Receptionist Beryl
* Go to talk to 3 NPCs
* There are 2 in main office, one in Misc
* Got to Reception - interact with laptop
* Choose Yes for allocating people
* Choose Swodniw for allocating Cathy	
* Choose Xunil for allocating Owen
* Choose Xunil for allocating James
* Choose Hsotnicam for allocating June

**Part 3: Find mentor**

* Got to Sam’s Office
* Talk to Sam
* Go to Reception and talk to Sophia
